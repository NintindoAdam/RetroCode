RetroCode

Visual Studio Code, dressed in Windows XP “Luna”.

A genuine, fully-working code editor with the soul of 2001. RetroCode is a fork of VSCodium with the classic XP blue chrome, glossy tabs, and Tahoma type baked right in. No extensions, no settings to tweak. Open it and you’re home.

Download for Mac macOS · Apple Silicon · 223 MB Download for Windows Windows 10/11 · x64 · 137 MB

Mac build is signed & notarized. Windows is unsigned (one-time SmartScreen prompt).

About _▢✕

RetroCode is built on VSCodium (the freely-licensed Code OSS build of Visual Studio Code) and uses the open-source Open VSX extension registry.
